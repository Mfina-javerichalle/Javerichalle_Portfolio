>Logiciels Utilisés

> Voici les différents outils et logiciels que j’ai utilisés lors de ma formation.

Visual Studio Code

L’éditeur de code complet de Microsoft, polyvalent avec de nombreux langages de programmation.

PHP

Langage de programmation libre, principalement utilisé pour produire des pages web dynamiques.

XAMPP

Logiciel permettant de mettre en place un serveur Web local pour exécuter des applications PHP/MySQL.

Microsoft Project

Logiciel de gestion de projet permettant de planifier, suivre et gérer efficacement les tâches et ressources.

MySQL

Système de gestion de base de données relationnelles libre et populaire.

GitHub & BitBucket

Plateformes de versionnage et de collaboration pour le développement de logiciels et sites web.

Eclipse

Environnement de développement intégré (IDE) populaire pour Java et d’autres langages.

Anaconda

Distribution Python complète pour la science des données, la programmation et l’analyse.

Lucidchart

Outil en ligne pour la création de diagrammes, schémas UML et cartes conceptuelles.
