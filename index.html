Veille Technologique

Langages & Frameworks : Évolution, Tendances et Enjeux

Ma veille porte sur l’évolution des langages de programmation et des frameworks, essentiels dans la création d’applications modernes, performantes et évolutives. Voici un aperçu de leur histoire, de leur place actuelle et de leur impact sur le développement web.

Les Origines des Langages & Frameworks

1970 - 2010
  • 1972 – C
    Le langage C devient la base de nombreux langages modernes (C++, Java…)
  • 1995 – Java & JavaScript
    Java révolutionne le back-end, JavaScript s'impose pour le web.
  • 2005 – PHP & l’essor des CMS
    Création de sites dynamiques avec WordPress, Joomla…
  • 2009 – Naissance de Node.js
    JavaScript arrive côté serveur et ouvre de nouvelles possibilités.

Les Langages & Frameworks Aujourd’hui
